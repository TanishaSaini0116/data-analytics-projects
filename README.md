# Hollywood Movie Profitability Analysis

Data Analytics portfolio project — Tableau dashboard analyzing Hollywood film data (2006–2012).

## 🚀 Live Site
Deploy via GitHub Pages (Settings → Pages → Deploy from main branch)

## 📁 Files
- `index.html` — Main portfolio page (all CSS + JS inline)
- `project_report.pdf` — **Replace this with your actual PDF report**

## 🔧 Setup
1. Upload all files to GitHub repo
2. Enable GitHub Pages in repo Settings
3. Replace `project_report.pdf` with your real PDF file
